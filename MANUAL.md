# かしこい献立 使い方マニュアル

## アプリの流れ（フロー図）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        かしこい献立 アプリの流れ                              │
└─────────────────────────────────────────────────────────────────────────────┘

  [初回セットアップ]
        │
        ▼
  ┌─────────────┐
  │   設定      │ ← 何人分・月予算・好き/苦手/アレルギー・食生活モードを設定
  └──────┬──────┘
         │
         ▼
  ┌─────────────┐      ┌─────────────┐
  │  冷蔵庫     │ ←─── │   チラシ    │ 食材を登録（手動 or チラシ撮影）
  └──────┬──────┘      └─────────────┘
         │
         ▼
  ┌─────────────┐
  │   ホーム    │ 「今日の献立候補を見る」or「冷蔵庫の食材で献立提案」

         │
         ▼

  │ 献立選択    │ 朝食・昼食・夕食から好きなメニューを選択

         ▼

                 ┌─────────────┐
                 │   家計簿    │ 支出を記録・確認
                 └─────────────┘
```

## 各画面の使い方

### 1. ホーム
- **今月の食費**：当月の支出と残り予算を確認
- **今日の献立候補を見る**：AIが朝・昼・夕の献立を提案
- **冷蔵庫の食材で献立提案**：冷蔵庫にある食材を優先した献立を提案
- 献立をクリックするとレシピサイト検索が可能

### 2. 冷蔵庫
- カテゴリ（肉類、魚介類など）から食材を選んで追加
- 分量を入力可能
- 登録した食材は献立提案時に優先的に使用

### 3. チラシ
- チラシを撮影 or ファイル選択で価格を読み取り
- 気になる商品を買い物リストに追加

### 4. 買い物リスト
- 献立から自動生成された食材一覧
- **チェック**で家計簿に記録するものを選択
- **「選択分を家計簿へ」**：チェックした商品だけ家計簿に記録
- **「全リストを買い物完了として記録」**：全てを一度に記録

### 5. 家計簿
- 月ごとの支出を確認
- 買い物リストから記録された支出が表示される

### 6. 設定
- **何人分？**：献立の人数（1〜10人）
- **月の食費予算**：月間の食費予算を入力
- **食生活モード**：バランス型・ダイエット・筋トレ・健康志向・ファミリー
- **食材の設定**：
  - **好き**：よく使いたい食材
  - **苦手**：避けたい食材
  - **除外**：アレルギーで使えない食材
- タブを切り替えても選択中のタブは保持されます

## よくある流れの例

### パターンA：献立を決めてから買い物
1. 設定で人数・予算・好みを設定
2. ホームで「今日の献立候補を見る」
3. 朝・昼・夕の献立を選択
4. 「買い物リストを作成」をタップ
5. 買い物リストで必要なものを確認・チェック
6. 「選択分を家計簿へ」で記録

### パターンB：冷蔵庫の食材を活かす
1. 冷蔵庫に食材を登録
2. ホームで「冷蔵庫の食材で献立提案」
3. 提案された献立から選択
4. 足りない材料が買い物リストに追加される

### パターンC：チラシの特売を活用
1. チラシ画面でチラシを撮影
2. 読み取った商品を買い物リストに追加
3. 家計簿に記録
